Online music project Which will stream music online using Web Sockets so that it is streamed
#Important Guidelines
You need to install Git, node and npm  for this project.
Clone this repository first
and type following commands.

npm install.

This command will install all dependencies required.

Then you can type
 npm start
 or
node index.js

This will be running on port 3000.
